==============================
Partner contact old acc number
==============================

This module allows you to store the old account numbers of customers and vendors.


Installation
============

To install this module, you need to:

#. Only install


Configuration
=============

To configure this module, you need to:

#. Anything


Usage
=====

To use this module, you need to:

#. Go to the Contacts module
#. Select or create a new contact
#. Go to the Invoicing tab
#. Fill in the fields 'Old customer account number' or/and 'Old vendor account number'.
